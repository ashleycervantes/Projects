*   **How to Run:** Execute via RStudio after downloading the full `MLB-Dashboard` directory.

### üêç College Leasing Google Review Analysis
**Stack:** `Python` | `Pandas` | `NLP`

*   **Key Features:** Sentiment analysis, comparative university benchmarking, and UIUC-specific trend identification.
*   **How to Run:** Ensure Python 3.x is installed. Navigate to the folder and run the `.py` script or Jupyter Notebook.
